## 0. 一句话方案与 signature

把「每点带环持续脉动、满屏同权、像带环的土星一直动很吵闹」的标记,重构为**一片站立在地表、静止不动、只靠体积与光说话的行星发动机式光柱**——**高的是重要的,亮/热的是刚发生的,其余安静地立着**。

- **本批唯一 signature = 静态体积光柱的「根部白热 → 柱身分类色 → 尖端消散」辉光梯度**(planet-engine bloom)。这是 D27「掌控全球的科技感」与 D3「宁静」的调和点:体积感与光给科技感,**全静止**给宁静。参照 cobe 的「定向光 + Phong 柔和辉光」质感,不复刻其点阵脸。
- **其余一律克制**:severity 只用**柱高**分档(不再用尺寸、不再用脉冲);新鲜度只用**辉光热度**编码;色相锁死分类语义;远景聚合、近景才展开;唯一残留的「动」是增量呼吸(登/离场),reduced-motion 下连它也瞬切。
- **交付前已「减一件」**:见 §5 —— 初稿给登场配了「柱体从地面升起」的高度动画,修订中**减为可选**(baseline 只保留呼吸 alpha),使全系统唯一的动仍是那一次增量呼吸。同时新鲜度**只用发光热度、不叠加去饱和**(预先不加第二通道)。

> M2/M4 如何被本方案吃掉:M2(硬边靶环)→ 光柱用**径向高斯软边 + 尖端软消散**,无任何硬轮廓几何(RingGeometry 退场),从根上消除「reticle」读法;M4(静止不可读新鲜度)→ **发光热度**(加性 bloom 强度)按事件年龄静态衰减,静止画面即可读「多新」,且与 severity(柱高)正交、不打架。

## 2. 度量基准与通道分配(方案骨架)

**度量基准**:全部尺寸以**球半径 R = 1.0** 为单位(SPEC-3.1)。光柱为**世界空间**物体(高度以 R 计),随相机远近**自然缩放**——远景小、近景大,天然贴合「植根于地表的物理体」的语感,并天然承担一部分 LOD。

**通道分配(正交、可 glance 拆读)——本卡的中枢约束**:

| 视觉通道 | 编码变量 | 是否受其它变量干扰 | 说明 |
| --- | --- | --- | --- |
| **柱高 H** | **severity** | 不受新鲜度/呼吸干扰(唯一例外:可选登场高度缓入,复用呼吸包络) | severity 的**主锚**;高度判读近线性,最可靠 |
| **发光热度**(加性 bloom 强度 / 白热核 / 大气散射) | **新鲜度**(age) | 基线由 severity 定档,age 只做乘子 | 「刚发生的在发烫」 |
| **整体 alpha**(含柱身+辉光+根部足印) | **呼吸态**(登场/离场) | SPEC-3.11 增量语义,**不改** | 只表达增量登/离场,与 severity/新鲜度无关 |
| **柱身 hue**(不透明基色) | **分类**(category) | SPEC-3.7 pin,禁改 | 身份锚,任何年龄/档位都读得出是哪类 |

关键区分(防三通道互相污染):**「柱身基色」与「辉光热度」是两层**。柱身 = SPEC-3.7 pin 的 category×sev hex,alpha 由呼吸驱动,**与年龄无关**——保证再旧的事件也读得出分类与 severity 高度,不会因变旧而消失(消失是**过期离场**的事,不是新鲜度的事)。辉光热度 = 骑在柱身上的加性 bloom,**由年龄调制**——刚发生=强 bloom(发烫),旧=弱 bloom/无白热核(冷却的余烬)。

## 初稿(Pass 1)

### 3.1 光柱形态(等径、高度阶梯、软边)

**等径**:所有 severity 共用同一根部半径,severity 只改高度(D27 锁定「等径柱、高度阶梯」)。

| 参数 | 取值(R=1) | 说明 |
| --- | --- | --- |
| 根部半径 `r0` | **0.010 R** | 全 severity 恒定;约 1% 半径的细基座 |
| 尖端半径 `r1` | **0.35 · r0 ≈ 0.0035 R** | 向上锥收束(尾焰感),非直筒 |
| 径向软边 | **高斯 α**,σ ≈ 0.55·局部半径,边缘渐隐至 ~0 | **消除硬轮廓**(直接修 M2 靶环) |
| 柱高 `H` sev1 | **0.05 R** | 矮桩,最安静 |
| 柱高 `H` sev2 | **0.09 R** | 中 |
| 柱高 `H` sev3 | **0.15 R** | 最高,天际线主角(比 1:1.8:3.0) |

- **几何取向**:柱沿**径向**(球心→地表点)向外站立。子相机点正下方的柱指向观察者、透视压扁为一枚发光足印(靠 §3.2 根部足印兜底可读);越靠临边(limb)柱越显全侧影、站出球缘射入太空——那正是参考图的「money shot」(见 §3.8)。
- **实现取向(参照,不强制)**:轴锁 billboard(绕径向轴朝相机旋转的软体积面片)或极简锥几何皆可,**目标是软体积、无硬边**;须 instancing 友好(SPEC-3.8:柱=实例化四边面片,逐实例携带 高度/色/发光/年龄 属性,梯度在 shader 里算)。billboard 与几何的选择属实现自由度。
- **尖端处理**:顶部 20% 用 smoothstep 软消散到全透明,**无硬尖端**(否则又成一根「立着的 reticle」)。

### 3.2 宁静静态辉光(色 / 强度 / 衰减 / 与大气·夜面叠加)

辉光是本批 signature,须**静、宁、有体积**。判据:静止画面里像「能量在根部堆积、向上散入太空」,不像「一根发光棒」也不像「一团过曝白」。

**(a) 色 —— 三段梯度(hue 仍取分类色,不改 SPEC-3.7)**

| 段 | 位置 | 色 | 意图 |
| --- | --- | --- | --- |
| 根部白热核 | h∈[0,0.12] 且 r∈[0,0.30·r0] | `mix(分类hex, 白热tint, w)` | 能量堆积的「热核」;体积感来源 |
| 柱身 | 核以上、软边以内 | **SPEC-3.7 pin 的 category×sev hex(原样)** | 分类+severity 身份,不动 |
| 尖端 | 顶部 20% | 分类 hex + α 消散→0 | 散入太空,保 hue 消 alpha |

- **白热 tint 不用纯白**(纯白刺、失宁静、丢分类偏向):暖色相(disaster/conflict/humanitarian)用**暖白 `#fff0e0`**,冷色相(news/launch/flight)用**冷白 `#e8f2ff`**。
- **白热核权重 `w = 0.65 · sev_factor · fresh_factor`**(sev_factor: sev3=1/sev2=0.45/sev1=0)。故**只有够新的高 severity 才真正白热**;sev1 或已冷却事件无白核、只有分类色。白核被限制在最内 30%·r0、最低 12% 高度——**柱身仍是分类色,分类身份不被白核吃掉**。

**(b) 沿高度衰减曲线(根亮尖暗)**

- 发光强度 `E(h) = E_sev · exp(-2.3 · h) · (1 − smoothstep(0.80, 1.0, h))`,h∈[0,1] 归一化高度。
  - 根 h=0 → `E_sev`;中 h=0.5 → ≈0.32·`E_sev`;顶经软消散干净归 0。
  - 指数衰减给「根部堆积、快速转柔、长尾散开」的体积感;末端 smoothstep 保证无浮空硬端。

**(c) 径向衰减**:α 高斯(§3.1 软边),中心实、边缘散——柱有「芯」有「晕」,读作 bloom 不读作实心棒。

**(d) 根部足印 bloom(落地辉光 / 远景与压扁态的兜底标记)**

- 地表上一枚软径向盘,是「此处有事件」的恒定锚点,也是远景 LOD 与子相机点压扁时的主可读体。
- 足印半径 `R_foot`:sev3 **2.4·r0**、sev2 1.8·r0、sev1 1.2·r0(随 severity)。
- 足印强度 = `E_foot_sev · fresh_factor`,低、加性但**clamp**。

**(e) 与大气壳 / 夜面 / 过曝的叠加关系(防 Dark Acid #1 与过曝白)**

- **混合模式 = screen(或加性 + 软 clamp),禁纯加性**。纯加性在密集/临边会叠成纯白(过曝白)。
- **合成发光亮度天花板 Y ≤ 220**:白热核峰值封顶在近白(带分类冷暖偏),**永不到 255 纯白**;多柱重叠不叠到白。
- **不靠近黑底**:柱立在批次一已建的**被照亮球面**上(昼底 Y≈63、夜底 Y≈23、太空 Y≈9),不是纯黑幕——这是破 Dark Acid 的结构前提。柱须在两半球都读得出:夜面靠亮度对比,昼面靠 hue 对比(见 §3.7 可读性地板)。
- **与大气(SPEC-3.4 `#4a90d9`)**:柱的 bloom **深度上叠在大气之上**(不遮标记原则不变);临边处柱下段与蓝色 halo 重叠,用 screen 混合避免「红柱 × 蓝晕」互相加亮成脏白(见 §3.8)。
- **与夜面海岸辉光(`#3a5a72` 低强度)**:柱远比其亮,不冲突;夜面反而是柱最出彩的背景。

### 3.3 常驻态静默:去 sev3 持续脉冲后,severity 怎么表达

**删除 SPEC-3.7「severity 3 必须有持续脉冲环」**(与 D27 直接冲突,BUG-031),severity 改由**两条静态通道冗余编码**:

| 档 | 柱高 H | 根部足印半径 | 根部发光基线 `E_sev`(fresh 时峰值 Y 参考) | 白热核 | 脉冲/动画 |
| --- | --- | --- | --- | --- | --- |
| sev3 | 0.15 R | 2.4·r0 | 强(≈210,封顶 220) | 有(强) | **无(删)** |
| sev2 | 0.09 R | 1.8·r0 | 中(≈150) | 弱 | **无** |
| sev1 | 0.05 R | 1.2·r0 | 弱(≈105) | 无 | **无** |

- 全部**静止**。severity 主读**柱高**(一眼分高矮),辅读**发光基线/白热核**(冗余,更抗遮挡与远近)。
- 破「满屏同权高饱和点」:sev1 是矮桩弱辉、sev3 才是高柱强辉,深底上「多安静矮桩 + 少数高热柱」。

### 3.4 相机 LOD 三档 + 密集区聚合(D24 / D23 L0 / 修 AES-001 M3)

以相机距 `d`(R=1)分三档;柱为世界空间物体已自然缩放,LOD 额外管**显隐与聚合**,保证每档 3 秒读出「哪里·什么类·多新」而不吵。

| 档 | 距离 d(R=1,可达区间 [1.8,6]) | 柱显隐 | 聚合 |
| --- | --- | --- | --- |
| **远** | d ≥ 3.0(含初始 3.2) | 以**根部足印为主**;sev1 收为足印点(不出全柱),sev2/sev3 出短柱 | 角距 < **5°** 的事件聚为一柱 |
| **中** | 2.4 ≤ d < 3.0 | 全 severity 出全柱,高度阶梯完整可读 | 仅极密子簇聚合(角距 < **2.5°**) |
| **近** | 1.8 ≤ d < 2.4(1.8 = SPEC-7.2 变焦下限) | 全部独立全柱、体积细节 | 基本不聚(角距 < **0.8°** 才聚) |

> **R-1 修订 2026-07-22**(承 REV-017 §3 整改点):三档距离分界重算进 SPEC-7.2 相机可达区间 **[1.8, 6]**(原「近 d<1.8」在 [1.8,6] 内不可达、其 0.8° 聚合永不触发、子句静默蒸发,违 CLAUDE.md §5.2)。修订采**三个可达档**——把原「中档」区间 [1.8,3.0) 拆为 **中 [2.4,3.0) + 近 [1.8,2.4)**,**远档边界 3.0 不动**(初始 3.2 仍落远档、「overview 足印为主」的已仲裁行为不变),三个角距阈值 5°/2.5°/0.8° 沿用、原样重挂到三个可达档。中/近分界 **d2=2.4** 取「球体角径恰好铺满 45° fov」的视觉临界(d=2.4 时球张角 ≈49°、刚溢出视野):远档球体完整落在视野内(overview)、中档球体填满至微溢(大洲级)、近档球体强溢出(区域/国家俯冲)。**取舍理由**:优先保三个可达档而非「二档+预留」——最近景 d=1.8 是用户明确看重的「近看细节/掌控感」美学重点,此处必须落**近档 0.8° 细聚合**(不并掉相邻事件);若让中档 2.5° 覆盖 d=1.8,会把相邻事件并粗、损失掌控感。近档不再是「预留」而是即刻可达可测(qa 可把相机拉到 [1.8,2.4) 验 0.8° 聚合),无子句蒸发。

**密集区聚合策略(不叠罗汉)**:

- 触发:同 LOD 下事件角距 < θ_LOD(或等价:根部足印在屏幕空间将重叠)即并为一簇。
- **代表柱**:取簇内**最高 severity** 成员定柱高(= max 成员高度),平手取**最新**成员;辉光热度取代表成员的 age。**掌控感语义**:一个区域的天际线由其「最重的那件事」定高——扫一眼天际线即知何处最严重。
- **hue 不做平均**(避免脏色):取代表成员的分类色,**不混合多类 hue**。
- 计数徽章/簇内展开 = 注记层(L1/label),**留注记批**,本卡 L0 只出一根代表柱。

### 3.5 增量呼吸兼容(不改 SPEC-3.11 / FM-09 呼吸本身)

**只说明光柱形态如何挂到现有 alpha 语义,不动呼吸时长/曲线**(D27 确认的正确基准):

- 光柱的 **master emission = 现呼吸 alpha**。呼吸值即整根柱(柱身+辉光+根部足印)的统一透明/发光乘子。
- **登场**:新柱 alpha 0→1 走**现有呼吸包络**——观感是「发动机点火、从暗到亮」。
- **离场**:过期/被替换柱 alpha 1→0 走同包络——「熄火、辉光退去」。
- **可选(复用同一包络、零新增时间参数)**:柱高随同一 alpha 包络 0→满(登场「升起」)/ 满→0(离场「沉回地面」),做「点火升起」的体积感。**若被判为多余的动**,退回 alpha-only(见 §5 减一件,baseline 即 alpha-only)。
- **不新增任何时间参数、不改呼吸曲线**;根部足印与柱身随同一 alpha 同步登离场。

### 3.6 reduced-motion 降级(P1,D27 明列必给)

OS `prefers-reduced-motion` 开启时:

- **增量呼吸 → 瞬切(snap)**:登场 alpha 0→1、离场 1→0 **瞬时完成,无淡入淡出、无高度缓入**;新柱直接以满高满亮出现,过期柱直接消失。
- **稳态无需额外冻结**:本批已把持续脉冲/常驻动画**整体删除**,静止态**本就全静**(这正是本批的净收益)。白热核是**静态高光**(无闪烁)、根部足印静态、新鲜度用静态发光编码——**全系统除那一次增量呼吸外无任何动**,故 reduced-motion 只需 snap 掉呼吸即完备。
- 自查:确认实现里**不存在**任何相机驱动的微闪、辉光呼吸、足印脉动等隐藏动画;若有,一并关。

### 3.7 静态新鲜度编码(修 AES-001 M4 / D23 L0「多新」)

**目标**:静止画面 3 秒读出「多新」,且与 severity(柱高)**正交不打架**。

- **承载通道 = 发光热度(加性 bloom 强度)**,不用高度(高度已归 severity)、不用 alpha(alpha 已归呼吸态)、不用柱身基色(基色 = 分类身份,须恒定可读)。
- **映射**:age `a = now − ts`,发光乘子 `F(a) = 0.5 + 0.5 · exp(−a / τ)`,**τ = 8h**,下限 **0.5**:

  | age a | F(a) | 观感 |
  | --- | --- | --- |
  | 0(刚发生) | 1.00 | 发烫,白热核满(若 sev 够) |
  | 1h | 0.94 | 仍很热 |
  | 4h | 0.80 | 温热 |
  | 8h | 0.68 | 转凉 |
  | 24h | 0.52 | 近余烬 |
  | ≥ ~1 天 | 0.50(地板) | 冷余烬:辉光弱、无白核,但柱身分类色与高度仍在 |

- `F` **只乘** {根部发光基线 `E_sev`、白热核权重、大气散射 bloom、根部足印强度};**不乘** {柱身 pin 基色、柱高、呼吸 alpha}。
- **正交性(不打架)**:severity 主读**柱高**(与 age 无关,任何年龄都清晰)→ 先判「多重要」;在同高度里,**更亮/更热者更新** → 再判「多新」。心智模型:**高=重要,热=新**。天际线一眼四类:高且热=重大且突发(最抓眼)、高且冷=重大但旧、矮且热=小事但刚发生、矮且冷=小且旧(背景)。
- **为何不再消失**:`F` 地板 0.5 且**只调辉光不调柱身**——旧事件仍是一根可辨的分类色柱,**离屏交给过期离场(SPEC-3.11),不交给新鲜度**(避免「变旧=看不见=丢事件」,承接批次一「安静不是消失」的教训)。

### 3.8 临边(limb)观感(自绘地球最易露怯的三处,逐一处理)

- **子相机点(柱指向观察者)**:透视压扁为近一枚点 → 靠**根部足印 bloom**(§3.2d)保证仍是一枚可读的分类色软盘,不塌成不可辨的线。
- **临边站柱(参考图 money shot)**:柱在球缘显全侧影、站出球缘射入太空——**保留、是卖点**;但须:①**深度遮挡**——球背面(过 limb)的柱被球体遮住,只留近侧;②近侧临边多柱在屏幕空间挤叠 → 用 **screen 混合 + 该档 LOD 聚合(θ_far 5°)** 防过曝与叠罗汉。
- **柱 × 大气 halo 混色**:临边柱下段压在蓝色菲涅尔 halo(`#4a90d9`)上,红/橙柱与蓝晕若纯加性会互相加亮成脏白 → **screen 混合 + 发光 Y≤220 封顶**;柱**深度叠在大气之上**(不遮标记不变),halo 只作背景晕,不与柱的芯抢。

## 5. 修订(Pass 2 → delta,减一件)

1. **减一件:登场「柱体升起」高度动画由必需降为可选**。baseline 定为 **alpha-only 呼吸**(§3.5);高度缓入仅作 dev 可选增强、复用同一包络、零新增时间参数,若验收觉多可直接砍。**理由**:signature 是「静而有体积」,应让全系统唯一的动收敛到「一次增量呼吸」,以最大化「宁静」;高度动画虽好看,是与「静」竞争的第二处动,先减。
2. **预先不加第二通道:新鲜度只用发光热度,不叠加「随龄去饱和」**。**理由**:去饱和会侵蚀柱身分类身份(hue 可读性),且属「多编一件」;发光热度单通道已足够 glance 读「多新」。(此为「减一件」的预防性执行,初稿即未纳入。)
3. **补白热核护栏文字**:白热核权重 `w = 0.65·sev·fresh`、限最内 30%·r0 与最低 12% 高度、封顶 Y≤220、冷暖白非纯白 —— 防其膨胀成霓虹管/过曝白(补 §4.1 #3/#1 残险)。
4. **补临边 screen 混合 + 发光封顶**,防柱×halo 脏白与多柱叠白(补 §4.2 临边、§3.8)。
5. **补可读性地板说明**:柱身基色 alpha 由呼吸驱动、**与年龄无关**,保证旧事件昼面靠 hue 对比仍可读、不因冷却掉进底面(补 §3.7)。

- **静止即宁静**:静止截图(不录屏)下,除增量呼吸外**无任何动**;稳态截图两帧无差异(脉冲/闪烁=Blocker)。
- **软体积、非靶环**:近景截图,柱须软边 bloom、无硬圆环/硬边轮廓(命中 M2 硬边=不达标)。
- **severity 高度阶梯可读**:同屏 sev1/2/3,柱高单调 sev3>sev2>sev1、一眼可分。
- **新鲜度可 glance**:构造 fresh 与 24h+ 同 severity 事件同屏,fresh 明显更热(bloom 强/有白核),旧者冷余烬但柱身分类色/高度仍清晰可读。
- **深底不 acid / 不过曝**:采样最亮像素 Y≤220、无纯白 255;非「满屏同权高饱和柱」(多矮冷桩 + 少高热柱)。
- **LOD/聚合**:远景密集真实震群不叠罗汉(聚为代表柱)、近景展开;三档切换密度受控。
- **临边**:limb 处站柱不塌、球背柱被遮、柱×halo 无脏白混色。
- **reduced-motion**:开启 OS 减弱动态后,登/离场瞬切、无淡入淡出、稳态全静。
