## 1. 设计决策（含 REV-013 强制的两项论证义务）

### 1.1 通道正交分配（本提案的中枢约束，承 aes 方案 §2）

层次挂在四条**正交**通道上、一通道一变量，任何两通道互不污染（可 glance 拆读）：

| 视觉通道 | 编码变量 | spec 落点 | 与其它通道的关系 |
| --- | --- | --- | --- |
| **柱高 H** | severity | SPEC-3.7a | severity 主锚；与新鲜度/呼吸/年龄无关 |
| **静态发光热度**（辉光基线 + 白热核） | 新鲜度（age） | SPEC-3.7a + 3.7b | 基线由 severity 定档，age（`F`）只做乘子 |
| **整体 alpha** | 呼吸态（增量登/离场） | SPEC-3.11（不改） | 与 severity/新鲜度无关 |
| **柱身 hue** | 分类（category） | SPEC-3.7 色相表（不改） | 身份锚；任何年龄/档位恒定可读 |

关键区分（防通道污染）：**「柱身基色」与「辉光热度」是两层**。柱身 = SPEC-3.7 pin 的 category×sev hex（不动），其 alpha 由呼吸驱动、**与年龄无关**——再旧的事件也读得出分类与柱高；**旧不等于消失**（离屏是 SPEC-6.3 过期离场的职责，不是新鲜度的职责）。辉光热度 = 骑在柱身上的加性静态 bloom，由年龄（`F`）调制。

### 1.2 【论证义务 A（REV-013 §9-1 强制）】删「severity 3 必须有持续脉冲环」硬句的一致性论证

REV-013 line 103/202 反复要求：动效批须给出「删/改『severity 3 必须持续脉冲环』与现硬 spec 的关系论证」。逐条如下：

**(1) 该硬句与 D27 正面冲突，必须删——非可选。**
D27（用户拍板）：「常驻标记不施加持续动画……呼吸动效只用于增量」。BUG-031 记用户实测否决常驻脉冲。SPEC-3.7 现句「severity 3 必须有持续脉冲环」是一条**常驻动画硬命令**，与 D27 语义直接对立。按 CLAUDE.md §7「上游/用户裁决与 spec 不符 → 走 spec 修改路径」，此句必须删除，不是弱化。

**(2) 该硬句原本服务的功能，已被两条静态通道冗余接管——删后功能不减反增。**
现文该句 + line 94「发光为最强权重杠杆：sev3 发光、sev1 为哑色小点」的**立法意图**是：让 sev3 从「满屏同权高饱和点」中跳出来（破 Dark Acid），即**severity 的可见性/显著性**。删除脉冲后，该显著性由两条**静态**通道冗余承载：
- **柱高阶梯**（SPEC-3.7a）：sev3 0.15R / sev2 0.09R / sev1 0.05R（比 1:1.8:3.0）——高度判读近线性、最可靠，一眼分高矮。
- **静态发光基线 + 白热核**（SPEC-3.7a）：sev3 强辉+强白核 / sev2 中辉+弱白核 / sev1 弱辉无白核。
二者**均为静止量**，联合表达「深底上少数高热柱、其余安静矮桩」，比单一脉冲更抗遮挡与远近。故删句后 severity 的显著性**增强而非削弱**，line 94 的立法意图由静态通道更好地兑现。

**(3) 为何「不再需要」脉冲：脉冲是与 D3 宁静竞争的手段，非 severity 的必要条件。**
脉冲从来不是 severity 语义的载体（severity 是离散三档，脉冲只是「让它显眼」的表现手段之一）。等径光柱方案证明：显眼可以靠**体积 + 光 + 高度**（静态）达成，无需靠动。D30（美学即护城河，判据「高级感 + 宁静 + 掌控感」）与 D3（界面本身须平静）要求把「动」收敛到最少。故删脉冲后不需要任何替代动画来「补」——静态通道已完备。

**(4) 删除后全系统唯一残留的「动」= 增量呼吸（SPEC-3.11），服务信息、非装饰。**
增量呼吸表达「新事件登场/过期离场」（D27 确认的正确基准），是信息态变化的反馈，不是常驻装饰；且在 reduced-motion 下瞬切（§1.3）。稳态截图两帧无差异——这正是本批相对现状的净收益。

**结论**：删「severity 3 必须有持续脉冲环」与 D27「常驻不施动画」完全一致；其原功能（severity 显著性）由柱高 + 静态辉光冗余接管且更强；删后无需替代动画。**删句是本提案的核心，不是副作用。**

### 1.3 【论证义务 B（REV-013 §9-1 强制）】reduced-motion（P1）条款

REV-013 §9-1 把「reduced-motion（P1）交互条款」列为动效批放行前置（现 spec 完全无此条款）。本提案新增 SPEC-3.11a：

- **稳态无常驻动画**：本批已整体删除持续脉冲/常驻动画，稳态本就全静（白热核为静态高光、根部足印静态、新鲜度用静态发光编码）；reduced-motion 下稳态无需额外冻结。
- **增量呼吸在 `prefers-reduced-motion` 下瞬切（snap）**：登场 alpha 0→1、离场 1→0 瞬时完成，无淡入淡出；新柱直接以满态出现、过期柱直接消失。
- **无隐藏动画**：实现须自查不存在任何相机驱动微闪、辉光呼吸、足印脉动等隐藏动画（若有一并关）。

此条同时是删脉冲（§1.2）的**闭环保证**：它把「稳态无动画」从「本批恰好如此」升格为**对外可验收契约**，防未来回填任何常驻动画。

### 1.4 新鲜度承载通道选择（为何是「发光热度」）

D23 L0「多新」须在静止稳态可读（修 AES-001 M4）。承载通道选择（不默选）：

| 候选通道 | 取舍 |
| --- | --- |
| **发光热度（加性静态 bloom）**（建议） | 三通道正交（§1.1）下唯一空闲的可 glance 通道；「刚发生的在发烫」符合直觉；只加辉光不动柱身，分类身份不受损。**取此。** |
| 高度 | 已归 severity（柱高），复用则两变量无法拆读。否。 |
| alpha/透明度 | 已归增量呼吸（登/离场）；复用会让「半旧事件」看着像正在离场（半淡出），语义打架。否。 |
| 柱身基色去饱和 | 侵蚀柱身分类身份（hue 可读性），且属「多编一件」（违减一件）。否。 |

**映射（对外可见契约，须 pin，否则 L0 变量静默蒸发）**：`F(a) = 0.5 + 0.5·exp(−a/τ)`，`τ = 8h`，下限 0.5；`F` **只乘辉光**（发光基线/白热核/散射/足印），**不乘**柱身基色/柱高/呼吸 alpha。地板 0.5 保证「变旧 ≠ 看不见」：旧事件仍是一根可辨的分类色柱，离屏交给 SPEC-6.3 过期离场，不交给新鲜度。

**正交性心智模型**：**高 = 重要（柱高，与年龄无关），热 = 新（辉光，与 severity 无关）**——天际线一眼四类：高且热=重大突发、高且冷=重大但旧、矮且热=小事刚发生、矮且冷=小且旧。

### 1.5 标记 LOD 与密集区聚合（修 AES-001 M3，与 SPEC-3.2a 严格区分）

- **对象辨析（防混条）**：本条 LOD 管的是**事件标记（光柱）对象**的显隐与聚合；SPEC-3.2a 的经纬网格/国界/地名 LOD（视觉第二批另案）管的是**底图注记要素**。二者是**不同对象**，SPEC-3.7c 与 SPEC-3.2a 互不覆盖、互不推导。
- **三档距离**（d 以 R=1 计，含初始 3.2）：远 d≥3.0（足印为主，sev1 收为足印点、sev2/sev3 出短柱）/ 中 1.8≤d<3.0（全 sev 出全柱）/ 近 d<1.8（全独立全柱）。
- **聚合角距**：远 <5° / 中 <2.5° / 近 <0.8°（同 LOD 下角距小于阈值即并为一簇）。
- **代表柱规则**：簇内取**最高 severity** 成员定柱高（高=max，平手取最新成员），辉光热度取代表成员 age；**hue 不做平均**（取代表成员分类色，防脏色）。掌控感语义：区域天际线由「最重的那件事」定高。
- **层级边界**：计数徽章/簇内展开属注记层（L1/label，留注记批）；L0 每簇只出一根代表柱（守 D23 L0 三变量封顶）。

### 1.6 质量判据分流（比照 SPEC-3.10/3.2a 先例）

对外质量项分两层落，避免 spec 过度 pin 脆弱像素判据（承 REV-013「反向过度 pin 已避免」）：
- **spec pin 关键落值**（对外契约、验收锚点）：混合模式 = screen（禁纯加性）；合成发光亮度天花板 Y≤220（禁纯白 255）；白热核体积上限（限最内 30%·r0 & 最低 12% 柱高）；白热 tint 用冷暖白非纯白。→ 入 SPEC-3.7a。
- **量测方法/像素判据交 testplan**（比照 SPEC-3.10「验收基准环境与量测方法由 testplan 定义」、SPEC-3.2a「采样几何/亮像素阈值属 testplan」）：如「无过曝纯白像素」「临边柱×halo 无脏白混色」「稳态两帧无差异」的采样几何、亮度定义、阈值等。→ 列入受影响面，qa 定义。

### 1.7 柱高 × 大气自洽（SPEC-3.4）

sev3 柱高 0.15R 使柱顶超出大气壳、射入太空（参考图 money shot 效果）。须与 SPEC-3.4「大气不遮挡标记」自洽：
- **深度关系**：光柱（含 bloom）在深度上叠于大气之上——延续 SPEC-3.4「不遮挡标记」原则不变（大气不遮柱）。
- **临边混色**：临边柱下段与蓝色菲涅尔 halo（`#4a90d9`）重叠处以 screen 混合 + Y≤220 封顶，避免「暖柱 × 蓝晕」纯加性叠成脏白；halo 仅作背景晕，不与柱芯抢。
- spec 层只需保证「柱可高出大气、大气不遮柱、临边不脏白」三点措辞与 SPEC-3.4 不矛盾；柱×halo 的具体混色实现（shader/深度写入）归 dev DP，本提案不写。

### 1.8 增量呼吸兼容确认（不改 SPEC-3.11，承 aes §8-6）

- 光柱 **master emission = 现呼吸 alpha**：呼吸值即整根柱（柱身+辉光+根部足印）的统一透明/发光乘子；登场 0→1、离场 1→0 均走**现有呼吸包络**，**不新增任何时间参数、不改 SPEC-3.11 呼吸时长/曲线**。
- 可选高度缓入（柱高随同一 alpha 包络 0→满登场）为 **dev 可选增强**、复用同包络、零新增时间参数；baseline = alpha-only（visual-batch2-motion §5 减一件）。此属实现自由度，不进 spec。
- **确认**：本提案措辞不与 SPEC-3.11/FM-09「呼吸只用于增量、正确基准不动」冲突；SPEC-3.11 正文不改。

### 2.2 新增 SPEC-3.7a 光柱形态与静态体积辉光

**现行**：无。

**建议**（新增于 SPEC-3.7 之后；单位 R=1，SPEC-3.1）：

> - **SPEC-3.7a 光柱形态与静态体积辉光**：事件标记为**世界空间**的等径体积光柱（随相机远近自然缩放），沿**径向**（球心→地表点）站立；无硬轮廓（RingGeometry 式靶环退场）。
>   - **形态取值**（单位 R=1）：根部半径 `r0` = **0.010 R**（等径，全 severity 恒定）；尖端半径 `r1` = **0.35·r0**（向上锥收束）；径向为**软边**（中心实、边缘渐隐至 ~0，无硬轮廓）；柱高 `H` sev1/sev2/sev3 = **0.05 / 0.09 / 0.15 R**（比 1:1.8:3.0）；尖端**顶部 20% 软消散至全透明**（无硬尖端）。
>   - **静态辉光**（本批 signature，须静、宁、有体积；全程无脉冲/闪烁）：沿柱高三段色梯度——**根部白热核 → 柱身 pin 基色（SPEC-3.7，原样）→ 尖端 α 消散（保 hue 消 alpha）**；发光沿高度衰减（根亮、尖暗，末端干净归 0）；根部另有**静态落地足印**（软径向盘，远景与子相机点压扁态的兜底可读体），足印半径 sev3/sev2/sev1 = **2.4 / 1.8 / 1.2·r0**。
>   - **白热核护栏（对外可见约束）**：白热核**仅 sev2/sev3 出现**（sev1 无白核）、sev3 强于 sev2，且随新鲜度冷却（旧事件无白核，见 SPEC-3.7b）；体积限于**最内 30%·r0 半径 & 最低 12% 柱高**内（不吞没柱身分类色）；tint 用**冷暖白非纯白**——暖色相（disaster/conflict/humanitarian）用暖白 `#fff0e0`，冷色相（news/launch/flight）用冷白 `#e8f2ff`。
>   - **发光强度档与合成护栏**：根部发光基线随 severity 递增，fresh 峰值亮度参考 sev3 ≈ Y210 / sev2 ≈ Y150 / sev1 ≈ Y105（相对档序 sev3>sev2>sev1 为硬约束，绝对值为量测参考，量测方法由 testplan 定义，比照 SPEC-3.10）；**混合模式 = screen（禁纯加性，防密集/临边过曝白）**；**合成发光亮度天花板 Y ≤ 220**（多柱重叠、白热核峰值均不到纯白 255）。
>   - **与大气/夜面自洽**：光柱（含 bloom）在深度上叠于大气之上，延续 SPEC-3.4「大气不遮挡标记」；临边柱段与菲涅尔 halo（`#4a90d9`，SPEC-3.4）重叠处以 screen 混合 + Y≤220 避免脏白；柱立于 SPEC-3.2a 被照亮球面上（非纯黑幕），两半球均须可读（夜面靠亮度对比、昼面靠 hue 对比）。
>   - 形态须与 **SPEC-3.8** instancing 约束兼容（≥200 标记不逐事件建 Mesh）；具体实现（billboard/几何/shader、精确衰减曲线形态与系数、高斯 σ、白热核权重系数）属实现自由度，归动效批 dev design-prompt，不入本条。

**理由**：把 D27 锁定的光柱形态与「静而有体积感的辉光」落成对外可见取值（aes visual-batch2-motion §6）；软边 + 尖端消散从根上消除 M2 硬边靶环读法（AES-001 M2）。对外可见落值（尺寸/柱高/足印/tint 色/白热核体积上限/Y≤220/混合模式/发光相对档序）入 spec；曲线函数形态与系数按 REV-013 改动 6「不 pin 衰减曲线形态」先例归 dev DP，避免过度 pin 脆弱系数。

### 2.3 新增 SPEC-3.7b 静态新鲜度契约

**现行**：无（D23 L0「多新」此前只靠增量呼吸表达、静止不可读，AES-001 M4 遗留）。

**建议**：

> - **SPEC-3.7b 静态新鲜度编码（D23 L0「多新」）**：事件年龄 `a = now − ts`（`ts` 见 SPEC-6.1，事件时间）经**发光乘子** `F(a) = 0.5 + 0.5·exp(−a/τ)`（`τ = 8h`，**下限 0.5**）调制标记的**辉光热度**——静止画面即可读「多新」，且与 severity（柱高）正交。
>   - `F` **只乘辉光**：根部发光基线、白热核权重、大气散射 bloom、根部足印强度（SPEC-3.7a）；**不乘**柱身 pin 基色、柱高、呼吸 alpha（SPEC-3.11）。
>   - 观感锚点：a=0（刚发生）→ F=1.00（发烫、白热核满，若 severity 够）；a=24h → F≈0.52（近余烬、辉光弱、无白核，但柱身分类色与柱高仍在）；a ≳ 1 天 → F=0.50 地板。
>   - **地板不为 0**：新鲜度**只调辉光不调柱身**，旧事件仍是可辨的分类色柱，**不因变旧而消失**——离屏交给过期离场（SPEC-6.3①），不交给新鲜度。
>   - 具体采样/量测方法（`now` 取值、`ts` 缺失兜底见 SPEC-6.1、断言阈值）由 testplan 定义（比照 SPEC-3.10）。

**理由**：这是 D23 L0 三变量「多新」的对外可见落地契约，必须入 spec，否则该 L0 变量在批次交界静默蒸发（承 AES-001 M4、CLAUDE.md §5.2「防子句蒸发」）。地板 0.5 + 只调辉光承接批次一「安静不是消失」的教训，避免「变旧=丢事件」。

### 2.4 新增 SPEC-3.7c 标记 LOD 与密集区聚合

**现行**：无（AES-001 M3 遗留：密集区叠罗汉）。

**建议**：

> - **SPEC-3.7c 标记 LOD 与密集区聚合**（修 AES-001 M3；**与 SPEC-3.2a 的经纬网格/国界/地名 LOD 为不同对象**——本条管事件标记对象的显隐与聚合，SPEC-3.2a 管底图注记要素，二者互不覆盖、互不推导）：以相机距 `d`（R=1，含初始 3.2）分三档，保证每档 3 秒读出「哪里·什么类·多新」而不叠罗汉。
>   - **远** d ≥ 3.0：以根部足印为主——sev1 收为足印点（不出全柱），sev2/sev3 出短柱；同 LOD 下角距 < **5°** 的事件聚为一簇。
>   - **中** 1.8 ≤ d < 3.0：全 severity 出全柱、高度阶梯完整可读；角距 < **2.5°** 才聚。
>   - **近** d < 1.8：全部独立全柱；角距 < **0.8°** 才聚。
>   - **代表柱规则**：簇内取**最高 severity** 成员定柱高（高度 = max，平手取**最新**成员），辉光热度取代表成员 age；**hue 不做平均**（取代表成员分类色，不混合多类 hue）。
>   - **层级边界**：计数徽章/簇内展开属注记层（L1/label，另案）；L0 每簇只出一根代表柱（守 D23 L0 三变量封顶）。

**理由**：D23 L0「密集区须聚合、不叠罗汉」与 D24 LOD 的对外可见落地；代表柱取 max severity 兑现「掌控感」（扫天际线即知何处最严重）；显式区分 SPEC-3.2a 防混条。

### 2.5 新增 SPEC-3.11a 减弱动态降级（reduced-motion，P1）

**现行**：无（spec 完全无 reduced-motion 条款；REV-013 §9-1 强制补）。

**建议**（新增于 SPEC-3.11 之后）：

> - **SPEC-3.11a 减弱动态降级（reduced-motion，P1）**：**稳态无常驻动画**——常驻标记（光柱/辉光/白热核/根部足印）与底图均为静态，稳态截图两帧无差异（无脉冲、无闪烁、无相机驱动微闪）。OS `prefers-reduced-motion` 开启时，**增量呼吸（SPEC-3.11）瞬切（snap）**：登场 alpha 0→1、离场 1→0 瞬时完成、无淡入淡出（含可选高度缓入亦瞬切）；新标记直接以满态出现、过期标记直接消失。**除该一次增量呼吸外，全系统不得存在任何常驻动画**（若引入须一并纳入本条降级）。

**理由**：兑现 REV-013 §9-1 强制的 reduced-motion P1 条款；把「稳态无动画」升格为对外可验收契约，闭环删脉冲（§1.2）并防未来回填常驻动画。与 D27「呼吸只用于增量」、SPEC-3.2①「初始无淡入」自洽。

2. **辉光取值 pin 粒度**（承 REV-013 §9-4/改动 6）：本提案 pin 形态尺寸/柱高/足印/tint 色/白热核体积上限/Y≤220/混合模式/发光相对档序 + 新鲜度 `F`，把精确衰减曲线函数与系数（exp 系数、smoothstep 端点、高斯 σ、白热核权重 0.65）归 dev DP。若 rev 判某系数（如衰减曲线形态）属对外可验收须 pin，则相应上移入 SPEC-3.7a；反之若判 fresh 峰值 Y 210/150/105 绝对值过脆应降为纯相对档序，亦可仲裁调整。已列取舍。
3. **新鲜度 `τ=8h` 与地板 0.5 的产品取值**：为 aes 锁定的对外可见阈值；若产品后续要调（如不同信源不同衰减速率），属 SPEC-3.7b 后续 spec 变更、另案。当前范围内按 aes 值 pin。
4. **柱高 0.15R 出大气壳的深度实现**：本提案在 spec 层只保证「柱可高出大气、大气不遮柱、临边不脏白」与 SPEC-3.4 自洽；具体深度写入/混色实现归 dev DP，若实现中发现与 SPEC-3.4 有难以调和的深度冲突，属实现反馈、回本提案或另登缺陷。
5. **判据演进建议（AES-review 判据，非本提案 spec 范围）**：aes §8 建议「深底上的事件色层次」判据补一句「硬边轮廓（RingGeometry 式 reticle）判不达标」，把 M2 教训固化进 AES-review 判据。此属 aes/rev 的验收判据演进，不在本 spec 提案范围，登记转 orch/aes。
6. **依赖裁决**：本提案须经 rev 仲裁通过后由 orch 应用（改写 SPEC-3.7 + 新增 SPEC-3.7a/b/c + SPEC-3.11a + §0 修改记录 + `make pin-spec` + 同步 testplan/FM + 后续 dev DP），dev 方可据此实现动效批；在此之前不得抢改 src/tests。
